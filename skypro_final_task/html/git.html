Git

  • init - расшифровывается как "initialization", что переводится как "Инициализация репозитория в существующем каталоге".
  • add - расшифровывается как "Добавить", "Команда, которая добавляет изменения из рабочего каталога в раздел проиндексированных файлов".
  • commit - расшифровывается как "Совершить", "Команда в системе контроля версий Git, которая фиксирует изменения в репозитории".
  • clone - расшифровывается как "Клон", "Команда, которая создаёт копию существующего репозитория в новую директорию".
